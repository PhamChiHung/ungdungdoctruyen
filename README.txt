# ungdungdoctruyen
Để cài đặt chương trình, bạn hãy làm theo các bước sau:
Bước 1: 
Cài đặt Android Studio:
	- Tải xuống và cài đặt Android Studio từ trang chủ của Android Developer: https://developer.android.com/studio
	- Làm theo các hướng dẫn và chờ trình cài đặt hoàn thành.
Bước 2:
Cài SDK:
	-Để mở Trình quản lý SDK trong Android Studio, hãy nhấp vào Tools > SDK Manager (Công cụ > Trình quản lý SDK) hoặc nhấp vào SDK Manager (Trình quản lý SDK)  trong thanh công cụ. Nếu không dùng Android Studio, bạn có thể tải các công cụ xuống bằng công cụ dòng lệnh sdkmanager.
	-Các gói bắt buộc:
	+Bộ công cụ xây dựng SDK Android
	+Bộ công cụ nền tảng SDK Android
	+Bộ công cụ dòng lệnh SDK Android
	+Nền tảng SDK Android
	-Nếu nhà mạng hoặc nhà sản xuất thiết bị đã lưu trữ tệp kho lưu trữ tiện ích bổ sung SDK trên trang web của mình, hãy làm theo các bước sau để thêm trang web đó vào Trình quản lý SDK Android:
        Nhấp vào thẻ SDK Update Sites (Trang web cập nhật SDK).
	Nhấp vào biểu tượng Add (Thêm)  ở đầu cửa sổ.
	Nhập tên và URL của trang web bên thứ ba đó rồi nhấp vào OK.
	Đảm bảo bạn đã chọn hộp đánh dấu trong cột Enabled (Bật).
	Nhấp vào Apply (Áp dụng) hoặc OK.
	Giờ đây, mọi gói SDK hiện có trên trang web đó sẽ xuất hiện trong thẻ SDK Platforms (Nền tảng SDK) hoặc SDK Tools (Bộ công cụ SDK) (nếu có).
Bước 3:
Tải xuống mã nguồn từ GitHub:
	- Tải xuống từ link github này:"https://github.com/PhamChiHung/ungdungdoctruyen.git"
	- Nếu bạn có liên kết GitHub của mã ngồn, hãy truy cập vào đó và tải xuống toàn bộ mã nguồn dưới dạng một thư mục nén.
	- Giải nén thư mục vào thư mục mà bạn muốn làm việc với mã nguồn.
Bước 4: 
Mở dự án trong Android Studio:
	- Mở Android Studio sau khi cài đặt thành công.
	- Chọn "Open an existing Android Studio project" trong màn hình chào đón hoặc chọn "File" -> "Open" trong menu chính để mở dự án.
	- Chọn thư mục chứa mã nguồn đã tải xuống và nhấn "OK".
Bước 5:
Cài đặt các phụ thuộc:
	- Android Studio sẽ tiến hành tải xuống và cài đặt các phụ thuộc (dependencies) của dự án từ kho lưu trữ Maven hoặc Gradle.
	- Chờ cho quá trình này hoàn thành.
Bước 6:
Xây dựng ứng dụng:
	- Nếu không tự động hiển thị, hãy chọn thẻ "Build" trong thanh bên cạnh "Project".
	- Nhấp vào nút "Make Project" (biểu tượng hình một cái đồng hồ bên cạnh biểu tượng "Play") để xây dựng ứng dụng.
	- Chờ cho quá trình xây dựng hoàn thành.
Bước 7:
Chạy ứng dụng:
	- Khi xây dựng thành công, trong thanh công cụ trên cùng của Android Studio, hãy kết nối trực tiếp điện thoại của bạn với máy tính.
	- Nhấp vào nút "Run" (biểu tượng hình một cây Android) hoặc nhấn phím Shift + F10 trên bàn phím để chạy ứng dụng.
	- Chờ cho quá trình cài đặt và chạy ứng dụng hoàn thành trên thiết bị di động của bạn.
